# Data Policy — периметр даних

1. Внутрішні дані (strw-state, код продуктів, метрики, листування) НІКОЛИ не надсилаються у зовнішні LLM/сервіси поза підключеними конекторами.
2. Генерація зображень/контенту зовнішніми інструментами — тільки з знеособленими даними.
3. Публічний web-пошук — дозволений; вставка внутрішніх фрагментів у пошукові запити — заборонена.
4. Секрети (ключі, токени) не пишуться в strw-state; тільки env/секрет-сховища.
5. Сумнів = ескалація в triage-inbox, тип `question`.
