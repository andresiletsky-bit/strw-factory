# Data Integrity Gate — верифікація даних і висновків (успадковано з grow-pm v1.13)

Обов'язковий для L2 Validation, L5 Portfolio і будь-якого артефакту з числами/трендами. Checker перевіряє КОЖЕН пункт; fail будь-якого = артефакт повертається maker'у.

## Чекліст
1. **Неповні періоди:** заборонена екстраполяція незавершеного місяця/кварталу як тренду. Кожне число — з явною анотацією періоду inline.
2. **Святкові/аномальні вікна:** порівняння YoY/MoM не через свята, розпродажі, force-majeure без явної примітки.
3. **Одноджерельні каскади:** висновок, що спирається на ОДНЕ джерело, не може породжувати ланцюг подальших тверджень. ≥2 незалежні джерела для ключових тверджень (TAM, попит, ціни конкурентів).
4. **Джерело кожної цифри:** лінк або «оцінка, метод: …». Цифра без походження = видаляється.
5. **Свіжість:** дані >12 міс для ринкових тверджень — позначаються як застарілі.
6. **Кореляція ≠ причинність:** причинні твердження вимагають механізму або експерименту.
7. **Публічність ≠ достовірність:** маркетингові матеріали конкурентів — джерело про позиціювання, не про факти.

## Формат анотації
`<число> (<джерело>, <період>, довіра: висока/середня/низька)`
